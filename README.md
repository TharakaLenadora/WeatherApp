# 🌤️ Weather Forecast Android App

A simple Android application that displays real-time weather forecasts using a public weather API. The app fetches weather data from the internet and presents daily weather information with icons, temperatures, humidity, and other weather details in a user-friendly interface.

---

## 📱 Features

- 🌍 Real-time weather information
- 📅 Daily weather forecast
- 🌡️ Temperature display
- 💧 Humidity information
- 🌬️ Wind speed details
- ☁️ Weather condition icons
- 📡 Fetches live weather data using REST APIs
- 📱 Simple and responsive Android UI

---

## 🛠️ Technologies Used

- **Java**
- **Android Studio**
- **Android SDK**
- **REST API**
- **JSON Parsing**
- **HTTPURLConnection**
- **Picasso** (Weather Icons)
- **XML Layouts**

## 🚀 How to Run

1. Clone the repository.

```bash
git clone https://github.com/TharakaLenadora/WeatherApp2.git
```

2. Open the project in **Android Studio**.

3. Sync Gradle dependencies.

4. Build and run the application on an Android device or emulator.
